*Daily A Noble Thought - May 26*

*Is it the Meditation? Is it the Introspection? Which helps for what?*

тЭУ Question :  Does Meditation help for Introspection?  Does Introspection help for Meditation?

ЁЯОп Answer :  Both helps each other. Introspection enables one to identify his own emotions, understand the mistakes committed on such emotions, conclude not to recur such things in future, implement such decisions.  Now, this introspection is included in spirituality by me. This psychological exercise can only rescue the man from the animal instinct in action.

Meditation helps to do such introspection successfully.  Like we search and find the objects in brightness, the mental peace provided by meditation helps to find the own shortcomings. Later, using the strength provided by such meditation, it can be eliminated too.  Similarly, if purified on introspection, meditation comes easily and nicely.  By using both, human life can be transformed into divine life. There is no other way.

*- Vethathiri Maharishi
